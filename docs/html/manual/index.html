Crystal Space Editing Environment

The CSEditing library is a set of tools and applications dedicated to the editing and the pre-processing of data content for the Crystal Space open-source 3D software development kit (SDK).

This document includes usage instructions, conceptual material, and detailed technical information.

This is edition 2.1.0 of the CSEditing manual for CSEditing version 2.1.0, last updated 20 March 2013.
